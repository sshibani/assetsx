# RenditionName

**Category:** Value set (enum)

## Description

`RenditionName` is the named size of a [Rendition](./Rendition.md). Each asset has at
most one rendition per name.

## Values

| Value | Description |
|---|---|
| `thumb` | Small thumbnail for grids/lists. |
| `standard` | Medium preview. |
| `large` | Large preview. |
| `original` | The untouched original (or a normalized copy). |
