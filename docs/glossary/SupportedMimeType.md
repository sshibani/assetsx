# SupportedMimeType

**Category:** Value set (enum)

## Description

`SupportedMimeType` is the allowlist of content types AssetX accepts for
[Asset](./Asset.md) uploads.

## Values

| Value | Description |
|---|---|
| `image/jpeg` | JPEG image. |
| `image/png` | PNG image. |
| `image/webp` | WebP image. |
| `image/gif` | GIF image. |
| `application/pdf` | PDF document. |
