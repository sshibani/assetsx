# AccountDTO

**Category:** Data transfer object

## Description

`AccountDTO` is the public representation of an [Account](./Account.md).

## Shape

| Field | Type | Notes |
|---|---|---|
| `id` | string | Account id. |
| `name` | string | Display name. |
| `slug` | string | Unique slug. |
| `status` | string | [AccountStatus](./AccountStatus.md). |
| `createdAt` / `updatedAt` | string (ISO) | Timestamps. |
